# E-commerce App - iti graduation project

Souq clone.

# Features

-Login and logout and authentication with firebase.</br>
-Fetching category and products from firestore.</br>
-Shopping cart.</br>
-Favourites.</br>
-Orders.</br>
-Customizable profile.</br>
-Dark mode.</br>
-Contact Us.</br>
